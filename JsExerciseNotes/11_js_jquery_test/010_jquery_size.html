jQuery 尺寸

jQuery 提供多个处理尺寸的方法

  • width() 获取元素的宽度
  • height() 获取元素的高度
  • innerWidth() 获取元素的内边距宽度
  • innerHeight() 获取元素的内边距高度
  • outerWidth() 获取元素的外边距宽度
  • outerHeight() 获取元素的外边距高度

jquery width() 和 height() 方法

width() 方法返回元素的宽度，height() 方法返回元素的高度。

语法：

  • $(selector).width()
  • $(selector).height()

示例：

这是一段文字

            
                <div id="box1" style="width: 200px; height: 100px; border: 1></div>
                <script>
                    var box1 = $('#box1');
                    console.log(box1.width()); // 200
                    console.log(box1.height()); // 100
                </script>

jquery innerWidth() 和 innerHeight() 方法

innerWidth() 方法返回元素的内边距宽度，innerHeight() 方法返回元素的内边距高度。

语法：

  • $(selector).innerWidth()
  • $(selector).innerHeight()

示例：

            
                <div id="box1" style="width: 200px; height: 100px; border: 1
                px solid #000000; margin: 5px;">
                <div id="box2" style="width: 150px; height: 80px; border: 1px
                solid #000000; margin: 10px 20px;"></div>
                <script>
                    var box2 = $('#box2');
                    console.log(box1.innerWidth()); // 150
                    console.log(box1.innerHeight()); // 80
                </script>

jquery outerWidth() 和 outerHeight() 方法

outerWidth() 方法返回元素的外边距宽度，outerHeight() 方法返回元素的外边距高度。

语法：

  • $(selector).outerWidth()

示例：

            
                <div id="box1" style="width: 200px; height: 100px; border: 1
                px solid #000000; margin: 5px;">
                <div id="box2" style="width: 150px; height: 80px; border: 1px
                solid #000000; margin: 10px 20px;"><p>这是一段文字</p></div>
                <script>
                var box2 = $('#box2');
                console.log(box1.outerWidth()); // 150
                console.log(box1.outerHeight()); // 42.5
                </script>
