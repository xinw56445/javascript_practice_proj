jQuery 获取并设置CSS 类

jQuery 操作CSS

  • addClass(class) - 向被选元素中添加一个或多个类
  • removeClass(class) - 从被选元素中删除一个或多个类
  • toggleClass(class) - 切换被选元素的class类
  • css(property,value) - 设置或获取被选元素的CSS属性

addClass() 方法

addClass() 方法向被选元素中添加一个或多个类。

语法: $().addClass(class)

参数: class - 一个或多个要添加的类，用空格分隔

实例:

给div h1 h2 p 标签添加class 属性样式

removeClass() 方法

removeClass() 方法从被选元素中删除一个或多个类。

语法: $().removeClass(class)

参数: class - 一个或多个要删除的类，用空格分隔

实例:

给div h1 h2 p 标签删除class 属性样式

toggleClass() 方法

toggleClass() 方法切换被选元素的class类。该方法对被选元素进行添加/删除类的切换操作

语法: $().toggleClass(class)

参数: class - 一个或多个要切换的类，用空格分隔

实例:

给div h1 h2 p 标签切换class 属性样式

css() 方法

css() 方法设置或获取被选元素的CSS属性。

语法: $().css(property,value)

参数: property - 要设置的CSS属性的名称

参数: value - 要设置的CSS属性的值

实例:

给div h1 h2 p 标签设置font-size 属性样式
