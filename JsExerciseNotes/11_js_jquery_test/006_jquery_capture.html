jQuery 获取内容和属性

jQuery 应用可以操作HTML 的属性和方法

jQuery DOM 操作

实用的DOM 操作的jquery 的方法

  • attr() 获取属性
  • prop() 获取属性
  • text() 获取文本
  • html() 获取html 内容
  • val() 获取表单元素的值
  • css() 获取css 样式
  • data() 获取data 属性
  • addClass() 添加class
  • removeClass() 删除class
  • toggleClass() 切换class
  • remove() 删除元素
  • empty() 清空元素
  • parent() 获取父元素
  • children() 获取子元素
  • siblings() 获取兄弟元素
  • find() 查找元素
  • eq() 获取元素
  • first() 获取第一个元素
