jQuery 选择器

jQuery 选择器允许对 HTML 元素或者单个元素进行操作

jQuery 选择器基于元素的id class type attribute attr_value 等查找

jQuery 中所有的选择器都以美元符号开头 $()

元素 选择器

jQuery 元素选择器基于HTML 元素进行定义

例如：

$(“p”) 选择所有p元素

$(“div p”) 选择所有div元素下的所有p元素

$(“div > p”) 选择所有div元素下的所有p元素

$(“div + p”) 选择所有div元素下的所有p元素，且p元素必须紧跟在div元素之后

$(“div ~ p”) 选择所有div元素下的所有p元素，且p元素必须紧跟在div元素之后

id 选择器

jquery id选择器通过 HTML 元素定义的id 属性指定元素

例如：

$(“#test”) 选择所有id为test的元素

class 选择器

jQuery class选择器 可以通过指定class 查找元素

例如：

$(“.test”) 选择所有class为test的元素

$(“.test1.test2”) 选择所有class为test1的元素下的所有class为test2的元素
