jQuery 事件

事件指的是什么

实例

  • 在元素上移动鼠标
  • 选取单选元素
  • 点击元素

jquery事件方法语法

$(selector).on(event,function(){})

例如：

                
                    // 3.7.1 写法
                    $("p").on("click",function(){
                        alert("你点击了p标签");
                    });
                    // 1.10.2版本写法
                    $("p").click(function(){
                        alert("你点击了p标签");
                    });

注意：引入jQuery 库版本不一样，库对应的方法也有所不一致，请使用固定版本

常见jQuery 事件方法

1. click()

click() 方法是按钮点击事件触发会执行的函数

                
                    $("p").click(function(){
                        alert("你点击了p标签");
                        });

2.dblclick()

dblclick() 方法是双击事件触发会执行的函数

                
                    $("p").dblclick(function(){
                        alert("你双击了p标签");
                        });

3.focus()

focus() 方法是元素获得焦点事件触发会执行的函数 blur()对应失去焦点事件触发的函数

                
                    $("p").focus(function(){
                        alert("你获得了焦点");
                        });

4. mouseleave()

mouseleave() 方法是鼠标离开事件触发会执行的函数

                
                    $("p").mouseleave(function(){
                        alert("鼠标离开了p标签");
                        });

5. mouseenter()

mouseenter() 方法是鼠标进入事件触发会执行的函数

                
                    $("p").mouseenter(function(){
                        alert("鼠标进入了p标签");
                    });

6. mousemove()

mousemove() 方法是鼠标移动事件触发会执行的函数

                
                    $("p").mousemove(function(){
                        alert("鼠标移动了p标签");
                        });

7. mouseover()

mouseover() 方法是鼠标移入事件触发会执行的函数

                
                    $("p").mouseover(function(){
                        alert("鼠标移入了p标签");
                        });

8. mouseout()

mouseout() 方法是鼠标移出事件触发会执行的函数

                
                    $("p").mouseout(function(){
                        alert("鼠标移出了p标签");
                        });
