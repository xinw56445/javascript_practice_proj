jQuery 简介

jQuery 是一个 JavaScript 的函数库

jQuery 是一个快速、简洁的 JavaScript 框架，是继Prototype之后又一个优秀的JavaScript代码库

jQuery 的设计思想是，写得更少，做得更快

jQuery 库包含以下功能 :

  • HTML 文档的遍历和操作
  • 事件处理
  • CSS 动画
  • Effects(高级特效)
  • Ajax
  • Utilities(实用工具)

jQuery 兼容性 : IE6+、Firefox、Chrome、Safari、Opera、iOS、Android

jQuery 下载 : https://jquery.com/

jQuery 安装

1. 网页中添加 jQuery

                
                    <p>官网下载</p>
                    <script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
                    <p>CDN(内容分发网络)下载</p>
                    <script src="https://cdn.staticfile.org/jquery/1.10.2/jquery.min.js"></script>

2. 本地下载 jQuery

将jQuery 库下载到本地引用

                
                    <script src="./jquery-3.3.1.min.js"></script>
