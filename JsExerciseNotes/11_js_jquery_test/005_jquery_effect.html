jQuery 效果

基础效果：隐藏、显示、切换，滑动，淡入淡出，以及动画

隐藏、显示

hide() 和 show()

语法：

$(selector).hide(speed,callback)

$(selector).show(speed,callback)

speed: 速度，可选，默认是400ms

callback: 回调函数，可选

                
                    $("p").on('click', function(){
                        $(this).hide();
                    });
                    $("p").on('click', function(){
                        $(this).show();
                    });

切换

toggle() 和 toggleClass()

语法：

$(selector).toggle(speed,callback)

$(selector).toggleClass(className,speed,callback)

speed: 速度，可选，默认是400ms

callback: 回调函数，可选

                
                    $("p").on('click', function(){
                        $(this).toggle();
                    });
                    $("p").on('click', function(){
                        $(this).toggleClass("highlight");
                    });

动画

animate() 方法允许您创建自定义的动画。

语法：

$(selector).animate(params,speed,callback)

params: 动画属性和值，可选

speed: 速度，可选，默认是400ms

callback: 回调函数，可选

                
                    $("p").on('click', function(){
                        $('p').animate({
                            left:'250px',  // p 标签内容向左偏移250像素
                        }, 1000);
                    });

                    // 可以操作多个属性
                    $("p").on('click', function(){
                        $('p').animate({
                            left:'250px',  // p 标签内容向左偏移250像素
                            fontSize: '24px',  // 注意这里必须使用Camel 标记法书写属性名，而不是font-size
                            width: '100px',
                            height: '100px'
                        }, 1000);
                    });

                    // 可以使用相对值
                    $("p").on('click', function(){
                        $('p').animate({
                            left:'+=250px',  // p 标签内容向左偏移250像素
                            fontSize: '-=24px',  // 注意这里必须使用Camel 标记法书写属性名，而不是font-size
                        });
                    });
                    
                    // 使用预定义的值
                    $("p").on('click', function(){
                        $('p').animate({
                            height: 'toggle',  // toggle 关键字表示高度从当前值切换到 0 或当前值， 还有show hide
                        });
                    });

                    // 队列功能操作连续动画
                    $("p").on('click', function(){
                        $('p').animate({
                                height: '300px', opacity: '0.4'
                            }, 1000).animate({
                                width: '100px', opacity: '0.8'
                            }, 1000).animate({
                                height: '300px', opacity: '0.4'
                            }, 100).animate({
                                width: '100px', opacity: '0.8'
                            });
                    });

停止动画

stop() 方法停止当前动画，并且将当前动画的位置重置为开始位置。

语法：

$(selector).stop(stopAll,goToEnd)

stopAll: 是否停止所有动画，可选，默认是false

goToEnd: 是否跳转到动画的结束位置，可选，默认是false

                    
                        $("p").on('click', function(){
                            // 停止动画
                            $("p").animate({

                            });
                        });
