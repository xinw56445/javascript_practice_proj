jQuery添加元素

添加新的 HTML 内容

  • append() - 在被选元素的结尾处插入内容
  • prepend() - 在被选元素的开头处插入内容
  • after() - 在被选元素之后插入内容
  • before() - 在被选元素之前插入内容

append() 方法

append() 方法在被选元素的结尾处插入内容。（仍然在该元素的内部）

语法：$(selector).append(content)

参数：

  • content - 要插入到元素中的内容。

实例：

在 <ul> 元素的结尾处插入一个新的 <li> 元素。

  • 这是第一个要点

after() 方法在被选元素之后插入内容。
