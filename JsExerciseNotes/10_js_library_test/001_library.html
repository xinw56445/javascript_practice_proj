JavaScript 库- jQuery、Prototype、MooTools

jQuery 是一个快速、简洁的 JavaScript 库，是继Prototype之后又一个优秀的JavaScript代码库。它 是兼容 CSS3 的，并且是可用来开发动态网页的。jQuery 还是一个开放源代码的项目，任何人都 可以为其开发和进步做出贡献。jQuery 库的官方网站是：www.jquery.com。

Prototype 是一套用于 JavaScript 的面向对象编程的框架，它是由 John Resig 编写的。 Prototype 最初是作为一个独立的 JavaScript 库，但现在已经成为一个全面的 JavaScript 库。 Prototype 最初的目标是成为一个轻量级的 JavaScript 库，但现在已经成为一个全面的 JavaScript 库。

MooTools 是一套用于 JavaScript 的面向对象编程的框架，它是由 John Resig 编写的。 MooTools 最初是作为一个独立的 JavaScript 库，但现在已经成为一个全面的 JavaScript 库。 MooTools 最初的目标是成为一个轻量级的 JavaScript 库，但现在已经成为一个全面的 JavaScript 库。
