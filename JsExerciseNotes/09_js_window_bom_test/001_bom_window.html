JavaScript Window 浏览器对象模型

浏览器对象模型（Browser Object Model）无正式标准

由于先已有的浏览器实现了Js 交互性方面的相同方法和属性，而此就是BOM的方法和属性

Window 对象

所有浏览器都支持window 对象，他表示浏览器窗口

所有JavaScript 全局对象、函数以及变量均自动成为window 对象的成员

document 是window.document

全局变量是window 对象的属性，全局方法是window对象的方法

Window 尺寸

window.innerWidth 窗口的内部宽度

window.innerHeight 窗口的内部高度

其他window 方法

window.open(url,target,features) 打开新窗口

window.close() 关闭当前窗口

window.alert(message) 显示一个带有一段文本消息和一个确认按钮的警告框

window.confirm(message) 显示一个带有一段文本消息和确认和取消按钮的对话框

window.prompt(message,default) 显示一个可提示用户输入的对话框，并在用户输入时返回输入的值

window.print() 打印当前窗口

window.scroll(x,y) 页面滚动到x,y坐标

window.moveTo(x,y) 移动当前窗口到x,y坐标

window.resizeTo(x,y) 调整当前窗口的大小到x,y
