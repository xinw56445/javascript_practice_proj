JavaScript Window Navigator

The navigator object contains information about the browser.

Window Navigator

navigator.appCodeName 浏览器代号

navigator.appName 浏览器名称

navigator.appVersion 浏览器版本

navigator.cookieEnabled 浏览器是否开启cookie

navigator.language 浏览器语言

navigator.onLine 浏览器是否连接到网络

navigator.platform 浏览器运行平台

navigator.userAgent 浏览器用户代理

特别注意： navigator 对象的信息具有误导性，不应该用于检测浏览器版本， navigator 数据可被用户更改 一些浏览器对测试站点会识别错误 浏览器无法报告晚于浏览器发布的新操作系统

浏览器检测

由于navigator 可以误导浏览器检测，使用对象检测可用来嗅探不同的浏览器

由于不同的浏览器支持不同的对象，可以使用对象来检测浏览器

例如只有Opera 浏览器支持属性"window.opera",可以根据if (window.opera) {...}识别出 Opera
