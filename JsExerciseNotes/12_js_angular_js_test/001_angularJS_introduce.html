AngularJS 简介

AngularJS 是 Google 开发的前端 MVC 框架，它可以让我们在浏览器端开发单页应用。

AngularJS 是一个Javascript 框架，通过指令扩展了HTML 。且通过表达式绑定数据到 HTML

AngularJS 是一个双向数据绑定框架，它可以自动检测数据变化，并更新 HTML 视图。

AngularJS 表达式 与 JavaScript 表达式

AngularJS 表达式是一个基于 JavaScript 表达式的简单模板语法。

AngularJS 表达式可以包含字母，操作符，变量

AngularJS 表达式可以写在HTML 中

AngularJS 表达式不支持条件判断，循环以及异常

AngularJS 表达式支持过滤器

AngularJS 扩展了HTML

  1. AngularJS 扩展了 HTML，通过指令来扩展 HTML 标签，并通过表达式绑定数据到 HTML 标签。
  2. AngularJS 可以克隆和重复 HTML 元素。
  3. AngularJS 可以隐藏和显示 HTML 元素。
  4. AngularJS 可以动态添加 HTML 元素。
  5. AngularJS 支持输入验证。
  6. ng-app 指令定义一个AngularJS应用程序
  7. ng-controller 指令定义一个AngularJS 控制器
  8. ng-model 指令绑定数据到 HTML 元素
  9. ng-bind 指令绑定数据到 HTML 元素
  10. ng-repeat 指令循环遍历数据
  11. ng-click 指令绑定事件到 HTML 元素

Hello {{name}}!

AngularJS 表达式

AngularJS 表达式可以绑定数据到 HTML 元素，AngularJS 会自动更新 HTML 元素。

AngularJS 表达式可以绑定事件到 HTML 元素，AngularJS 会自动调用事件处理函数。

  1. AngularJS 表达式写在双大括号里面: {{expression}}.
  2. AngularJS 表达式可以绑定数据到 HTML 元素。与ng-bind相似
  3. AngularJS 将在表达式书写的位置“输出数据”

实例：

Hello {{username}}!
