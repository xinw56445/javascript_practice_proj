AngularJS 事件

AngularJS 提供了事件处理的功能，我们可以使用指令来处理事件

ng-click 指令

ng-click 指令直接绑定 click 事件到HTML 上，类似于onclick 事件

Click me to toggle:

当你点击这个button，myCheck的值会改变，然后 disabled 状态也会改变

myCheck: {{checked}}

disabled: {{!checked}}

显示 HTML 元素

ng-show 指令可以设置应用隐藏或显示app内容

ng-show 接收的参数是布尔值，true 显示元素，false 隐藏元素

名: 姓:

姓名: {{lastName + " " + firstName}}

ng-hide 指令

ng-hide 指令隐藏或显示一个 HTML 元素。

ng-hide 指令直接绑定应用程序数据到 HTML 的style.display 属性上，其值为布尔值
