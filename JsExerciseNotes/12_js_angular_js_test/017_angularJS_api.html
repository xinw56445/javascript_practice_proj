AngularJS API(应用程序编程接口)

AngularJS 全局API 由于执行常见任务的JavaScript 函数集合

  • 比较对象
  • 获取元素
  • 事件处理
  • 日期和时间
  • 浏览器对象
  • 错误处理

通用API函数

函数 描述
angular.copy(source, destination) 将源对象的值复制到目标对象
angular.equals( obj1, obj2 ) 比较两个对象是否相等
angular.extend(destination, source) 将源对象的所有可枚举属性复制到目标对象
angular.injector(modules) 返回一个新的 AngularJS.injector 实例
angular.module(name, [requires], [configFn]) 创建一个新的 AngularJS 模块
angular.bootstrap(element, [modules]) 开始 AngularJS 应用
