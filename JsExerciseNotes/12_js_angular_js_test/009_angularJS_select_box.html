AngularJS 选择框

使用 ng-options 创建选择框

AngularJS 中可以使用ng-option 指令来创建一个下拉列表，列表项通过对象和数组循环输出

在下面的例子中，我们将创建一个下拉列表，列表项通过对象循环输出

请选择一个颜色：

你选择的颜色是: {{color.name}}

ng-options 与 ng-repeat

ng-repeat 也可以创建下拉列表

请选择一个颜色：

你选择的颜色是: {{cor}}

对比方式：

ng-repeat 指令是通过数组来循环 HTML 代码来创建下拉列表，但 ng-options 指令更适合创建下拉列表

ng-repeat 有局限性，选择的值是一个字符串

ng-options 指令可以将对象中的属性作为值，并将对象的属性作为显示值

数据源为对象

使用对象作为数据源的实例；

你选择的值为在 key-value 对中的 value

请选择一个颜色：

你选择的颜色是: {{cors}}

--value为对象进行选择的方法

请选择一个颜色：

你选择的颜色是: {{c.name}}

你选择的色号是: {{c.color}}

--value直接进行选择的方法

你选择的色号是: {{c.color}}
