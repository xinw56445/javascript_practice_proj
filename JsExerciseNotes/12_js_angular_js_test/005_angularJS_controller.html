AngularJS 控制器

AngularJS 控制器是用来处理应用程序逻辑的 JavaScript 函数。控制器通常包含应用程序的业务逻辑。

AngularJS 应用程序被控制器控制

控制器 ng-controller

在 HTML 中使用 ng-controller 属性来指定控制器。

在 ng-controller 属性中，指定要应用于元素的控制器的名称。

控制器是JavaScript 对象，由标准的 JS 对象的构造器创建

实例：

名：

姓：

全名：{{firstName + " " + lastName}}

控制器方法

控制器方法是控制器的公共函数。

在控制器中定义的方法可以访问控制器的属性和方法。

实例：

全名：{{fullName()}}

外部文件中的控制器

在大型的应用程序中，通常把控制器储存在外部文件中

通过script 标签引入到HTML 文件中
