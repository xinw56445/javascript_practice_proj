AngularJS 表单

AngularJS 表单是输入控件的集合

HTML 控件：

  • input
  • button
  • select
  • textarea

输入框（input）

ng-model 指令将应用数据与 HTML 控件进行了数据绑定

例如：当我们输入数据，我们会在页面上看到输入的数据

entered: {{myInput}}

ng-required 指令

ng-required 指令直接绑定应用数据到 HTML 的required 属性上

例如：当我们提交一个没有填的数据，我们会看到一个错误提示

输入不能为空!!!

isRequired : {{isRequired}}

myInput : {{myInput0}}

ng-pattern 指令

ng-pattern 指令直接绑定应用数据到 HTML 的pattern 属性上

例如：当我们输入的数据不符合要求，我们会看到一个错误提示

输入的不是字母!!!
