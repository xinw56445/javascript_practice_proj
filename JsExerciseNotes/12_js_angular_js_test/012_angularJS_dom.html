AngularJS HTML DOM

AngularJS 为HTML DOM 元素的属性提供了绑定应用数据的指令

ng-disable 指令

ng-disable 指令直接绑定应用程序数据到 HTML 的disabled 属性上

例如：当我们勾选 disabled 属性之后，我们发现与其绑定的button 无法点击了

disabled

myCheck: {{myCheck}}

myCheck: {{!myCheck}}

myCheck: {{myCheck? 'true' : 'false'}}

ng-show 和 ng-hide 指令

ng-show 指令隐藏或显示一个 HTML 元素。

ng-show 指令直接绑定应用程序数据到 HTML 的style.display 属性上，其值为布尔值

ng-show 实例：上午12点以后下面这句话是不可见的

当前时间：{{localTime}}

12点以后我就消失
