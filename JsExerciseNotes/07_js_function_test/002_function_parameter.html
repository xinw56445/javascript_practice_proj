JavaScript 函数参数

1。参数规则

函数定义不会为参数（parameter）规定数据类型

函数不会对所传的参数（argument）实行类型检测

函数不会检查所接收参数的数量

参数默认

        function myFunction(x, y) {
            if (y === undefined) {  // 如果调用参数的时候省略了参数，则设置为undefined
                y = 0;
            }
        }

2。arguments 对象

如果函数调用的参数太多（超过声明），则可以使用 arguments 对象来达到这些参数

arguments 对象包含函数调用时使用的参数数组

3。参数通过值传递

函数调用中的参数（parameter）是函数的参数（argument）

参数 通过值传递，函数只知道值，而不是参数的位置

如果函数改变了参数的值，它不会改变参数的原始值

参数的改变在函数外是不可见的

4。对象是由引用传递的

对象引用的是值

函数改变了对象属性，他也改变了原始值

对象属性的改变在函数之外是可见的
