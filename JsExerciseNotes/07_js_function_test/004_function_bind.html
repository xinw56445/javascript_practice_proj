JavaScript 函数绑定

1. 函数借用

bind() 方法一个对象可以从另一个对象借用方法

2. 保留this

有时候必须使用bind() 方法来防止丢失this

当函数做回调时，this会丢失

使用bind() 对函数绑定可避免

3. 什么是this

this 关键字引用对象

  • 在对象方法中，this引用该对象
  • 单独使用时，this引用全局对象
  • 在函数中，this引用全局对象
  • 在函数中，严格模式下，this是undefined
  • 在事件中，this引用接受事件的元素
  • call()、apply() 和 bind() 等方法可以将 this 引用到任何对象。
