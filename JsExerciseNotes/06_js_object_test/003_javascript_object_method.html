JavaScript 对象方法

1. 对象方法是能够在对象上执行的操作

          var person = {
            firstName: "Bill",
            lastName : "Gates",
            id       : 648,
            fullName : function() {
              return this.firstName + " " + this.lastName;
            }
          };

方法是包含函数定义的属性

方法是储存对象属性的函数

2. this 关键字

在Javascript 中，被称为this的事物，指的是拥有该代码的对象

this的值，在函数中使用时，是拥有该函数的对象

this 不是变量，是关键字，无法改变this的值

3. 访问对象方法

创建对象方法的语法 // methodName: function() {}

访问对象方法的语法 // objectName.methodName()

4. 添加新的方法

向对象添加方法是在构造函数内部完成的
