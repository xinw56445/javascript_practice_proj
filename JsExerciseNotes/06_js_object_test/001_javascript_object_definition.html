JavaScript 对象定义

1. 所有 JavaScript 值，除了原始值，都是对象。

  • 布尔是对象（如果用 new 关键词定义）
  • 数字是对象（如果用 new 关键词定义）
  • 字符串是对象（如果用 new 关键词定义）
  • 日期永远都是对象
  • 算术永远都是对象
  • 正则表达式永远都是对象
  • 数组永远都是对象
  • 函数永远都是对象
  • 对象永远都是对象

2. 对象属性

对象属性可以是原始值，其他对象以及函数

变量能够包含单个的值 var person = "Bill Gates"

对象也是变量，但对象可以包含很多值 var person = {firstName: "Bill", lastName: "Gates"}

对象是命名值的集合（命名值也被称为属性，比如firstName）

3. 对象方法

方法是可以在对象上面执行的动作

对象方法是包含函数定义的对象属性

var person = {firstName: "Bill", lastName: "Gates", fullName: function() {return this.firstName + " " this.lastName;}}

对象是被称为属性和方法的命名值容器

4. 创建对象

使用对象文字 var person = {};

使用关键字 new 创建 var person = new Object();

定义对象构造器，然后创建构造类型的对象

5. 对象是易变的

对象通过引用寻址，而非值

假设person是一个对象，var x = person;不会创建副本

对于x 他就是对象person 是同一个对象，对x的任何改变都会改变person
