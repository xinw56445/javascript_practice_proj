JavaScript 对象属性

1. Javascript 属性

访问对象属性语法

          objectName.property      // person.age
          objectName["property"]   // person["age"}
          objectName[expression]   // x = "age"; person[x]
          for in 循环遍历访问        // for (x in person) {txt += person[x];}

添加对象属性

          可以通过赋值向已存在的对象添加新属性  // person.xxx = "";
          不可以使用保留词作为属性名，方法名称都不可以

删除属性

          delete 关键词从对象中删除属性   // delete person.age;
          delete 关键词会同时删除属性的值和属性本身。

2. 属性值

所有属性都有名称，此外他们还有值

值是属性的特征之一，其他特征包括：可列举，可配置，可写
