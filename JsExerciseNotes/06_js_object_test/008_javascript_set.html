JavaScript Set

1. Set 是一组唯一值的集合，每个值只能在Set中出现一次/

Set 可以容纳任何数据类型的值

        new Set()     创建新的 Set。
        add()         向 Set 中添加新元素。
        delete()      从 Set 中移除元素。
        has()         如果值存在则返回 true。
        clear()       从 Set 中移除所有元素。
        forEach()     为每个元素调用回调函数。
        values()      返回包含 Set 中所有值的迭代器。
        keys()        与 values() 相同。
        entries()     返回迭代器，其中包含 Set 中的 [value,value] 值值对。
        size          返回Set 中元素的数量

2. 创建Set

将数组传给new Set()

创建一个新的Set， 使用add() 方法添加值或变量

如果add 添加了多个相同元素，则只会显示第一个

3. forEach()方法

为每个Set 元素调用函数

4. values() 方法

values() 方法返回包含Set中所有值的iterator对象

使用iterator 对象范围元素

5. keys()方法

Set 没有键 ，返回与values() 相同的结果

6. entries() 方法

entries() 方法返回的是[value, value]值值对
