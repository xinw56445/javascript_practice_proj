JavaScript this关键字

1. this是什么

this 关键词指的是它所属的对象

在对象方法中，this 指的是此方法的“拥有者”

单独的 this

在单独使用的时候，拥有者是全局，this指的是全局对象[object Window]

2. 函数中的 this

函数的拥有者默认绑定 this[object Window]

3. 事件处理程序中的this

HTML 事件处理程序当中， this指的是接受此事件的HTML元素

4. 对象方法绑定

this 对于对象来讲就是 就是对象自身

5. 显示函数绑定

call() 和 apply() 方法都可以用于将另一个对象作为参数调用对象的方法
