JavaScript 数学方法

可以在不首先创建Math对象的情况下使用所有方法和属性（常量)

1.圆周率 Math.PI

Math.PI 返回圆周率

2.Math.round()

Math.round(x)方法返回x四舍五入的结果，int

3. Math.pow()

Math.pow(x， y) 返回x的第y次幂

4. Math.sqrt()

Math.sqrt(x) 返会x的平方根

5. Math.abs()

Math.abs(x)返回x的绝对值

5. Math.ceil()

Math.ceil(x) 返回x的向上舍入最接近的整数

6. Math.floor()

Math.floor(x) 返回x的向下舍入最接近的整数

7. Math.sin()

Math.sin(x) 返回x的正弦值（角度）

8. Math.cos()

Math.cos(x) 返回x的余弦值(角度)

9.Math.min() 和 Math.max()

返回数列对象的最低值和最高值

10. 随机数

Math.random() 返回介于0-1之间的随机小数

Math.random() 与 Math.floor() 一起使用用于返回随机整数

定义一个用于返回指定范围的随机整数函数(含头不含尾)

Math 属性常量

          Math.E          // 返回欧拉指数（Euler's number）
          Math.PI         // 返回圆周率（PI）
          Math.SQRT2      // 返回 2 的平方根
          Math.SQRT1_2    // 返回 1/2 的平方根
          Math.LN2        // 返回 2 的自然对数
          Math.LN10       // 返回 10 的自然对数
          Math.LOG2E      // 返回以 2 为底的 e 的对数（约等于 1.414）
          Math.LOG10E     // 返回以 10 为底的 e 的对数（约等于 0.434）
