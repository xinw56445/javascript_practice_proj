JavaScript 循环

1. for 多次遍历代码块

        for (语句 1; 语句 2; 语句 3) {
            要执行的代码块
        }

语句1 在循环代码块开始之前执行

语句2 定义循环代码块执行的条件

语句3 在循环代码块每次被执行后执行

语句1 中可以初始化多个值

语句1 可以使用省略写法

语句2 是可选的，如省略，则必须在循环内break，否则就是死循环

语句3 是可选的，可做其他事，可以直接在循环中体现

2. for in 遍历对象属性

        for (key in object) {
            要执行的代码块
        }

对象属性遍历

数组遍历

forEach() 顺序遍历数组

3. for of 遍历可迭代对象的值

        for (variable of iterable) {
          // code block to be executed
        }

variable- 对于每一次迭代，下一个属性的值都会分配给变量

iterable- 具有可迭代属性的对象

遍历数组

遍历字符串

4. while 条件为Ture 时一直循环代码

        while (condition) {
            // 执行的代码块
        }

循环输出变量

5. do while 至少先执行一次代码再去判断条件

        do {
            执行代码块
        } while (条件)

循环输出变量

6. for 和 while 对比

for 和 while 循环相当类似，语句1 ，2都可以省略

7. break 和continue

break 语句跳出循环

continue 语句终断本次循环执行下一次循环
