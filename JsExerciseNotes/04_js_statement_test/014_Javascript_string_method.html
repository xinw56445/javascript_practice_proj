3. lastIndexOf 方法返回字符串指定文本最后一次出现的索引位置, 未找到文本均返回-1


4. 检索字符串中的字符串 search() 方法搜索特定值的字符串，并返回匹配的位置


5. 提取部分字符串

  • slice(start, end)
  • substring(start, end)
  • substr(start, length)

slice() 提取字符串的某个部分并在新字符串中返回被提取的部分


substring() 用法类似于 slice() 但是不支持负索引。


substr() 用法类似于 slice() param：start：起始索引，length：提取字符的长度


6. 替换字符串内容 replace()， 只能替换首个匹配


replace() 不区分大小写，正则写法
