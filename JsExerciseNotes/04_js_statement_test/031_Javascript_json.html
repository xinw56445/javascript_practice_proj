JavaScript JSON

1. 什么是json

JSON 是存储和传输数据的格式

JSON 经常在数据从服务器发送到网页使用

  • JSON 指的是JavaScript Object Notation
  • JSON 是轻量级的数据交换模式
  • JSON 独立语言 *
  • JSON 是"自描述的"容易理解

JSON 实例

        {
          "employees":[
              {"firstName":"Bill", "lastName":"Gates"},
              {"firstName":"Steve", "lastName":"Jobs"},
              {"firstName":"Alan", "lastName":"Turing"}
          ]
        }

2. JSON 语法规则

  • 数据是名称/值对
  • 数据是由逗号分隔
  • 花括号保存对象
  • 方括号保存数组
  • 最好使用双引号

3. JSON 对象和数组

对象可以包含多个名称/值对

数组可以包含多个对象

4. JSON 文本转换为 JavaScript对象

JSON 的通常用法是从 web 服务器读取数据，然后在网页中显示数据

使用parse()将字符串转换为JSON格式

用到的方法

函数 描述
JSON.parse() 将字符串转换为JSON格式
JSON.stringify() 将JSON格式转换为字符串

5. JSON 格式化

JSON 格式在语法上与创建 JavaScript 对象代码是相同的。

由于它们很相似，所以 JavaScript 程序可以很容易的将 JSON 数据转换为 JavaScript 对象。
