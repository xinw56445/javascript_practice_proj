JavaScript 错误 - Throw 和 Try to Catch

        try 语句使您能够测试代码块中的错误。

        catch 语句允许您处理错误。

        throw 语句允许您创建自定义错误。

        finally 使您能够执行代码，在 try 和 catch 之后，无论结果如何

1. try...catch

try 定义代码块以便检查错误，catch 处理错误

2. throw

throw 语句运行创建自定义错误，异常可以是字符串，数字，布尔或对象

4. 内置Error 对象

error 对象提供 name 和 message 属性

message 设置或返回错误消息

name 设置或返回错误名

          EvalError       已在eval()函数中发生的错误
          RangeError      已发生超出数字范围的错误
          ReferenceError  已发生非法引用
          SyntaxError     已发生语法错误
          TypeError       已发生类型错误
          URLError        在encodeURL()中已发生的错误

EvalError 后续更新版本用 SyntaxError

RangeError 会在您使用了合法值的范围之外的数字时抛出。

ReferenceError 引用尚未声明的变量则会报错

SyntaxError 代码编写有语法错误会报这个错误

TypeError 变量引用有类型错误会报这个错误

URLError 在 URI 函数中使用非法字符
