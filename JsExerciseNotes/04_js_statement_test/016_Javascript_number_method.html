JavaScript 数字

1. JavaScript 数值书写可以带小数点或不带

2. JavaScript 数值始终是64位的浮点数


3. 数值精度

  • 整数（不使用指数或科学计数法）会被精确到 15 位。

  • 小数的最大数是 17 位，但是浮点的算数并不总是 100% 精准：

4. 数字和字符串相加

  • 对两个数相加，结果将是一个数

  • 对两个字符串相加，结果将是一个字符串的级联

  • 一个字符串和一个数字相加，或是数字和字符串相加，都是字符串级联

5. 数字字符串

  • javascript尝试将字符串转为数字

6. NaN 非数值

  • NaN 属于 JavaScript 保留词，指示某个数不是合法数。 尝试用一个非数字字符串进行除法会得到 NaN（Not a Number）：

  • 只有字符串包含纯数值才可以进行

  • NaN 如果在数学运算中使用了 NaN 则结果也是NaN

7. Infinity

  • Infinity （或 -Infinity）是 JavaScript 在计算数时超出最大可能数范围时返回的值。

  • 除以 0（零）也会生成 Infinity

8. 十六进制

  • JavaScript 会把前缀为 0X 的数值上了解析为十六进制。

11. 把变量转化为数值

这三种 JavaScript 方法可用于将变量转换为数字：

  • Number() 方法
  • parseInt() 方法
  • parseFloat() 方法

这些方法并非数字方法，而是全局 JavaScript 方法

全局方法

JavaScript 全局方法可用于所有 JavaScript 数据类型

  • Number() 方法

  • parseInt() 方法

  • parseFloat() 方法

12. 数值属性

属性 描述
MAX_VALUE 返回 Javascript 中可能的最大数
MIN_VALUE 返回 JavaScript 中可能的最小数。
NEGATIVE_INFINITY 表示负的无穷大（溢出返回）。
NaN 表示非数字值（"Not-a-Number"）。
POSITIVE_INFINITY 表示无穷大（溢出返回）。
