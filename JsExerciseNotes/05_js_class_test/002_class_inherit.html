JavaScript 类继承

1. 类继承

使用 extends 关键字创建类继承

使用类继承创建的类继承了父类所有的方法

2. supper() 方法引用父类

通过在 constructor方法中调用 supper() 方法，我们调用了父级的 constructor 方法

从而我们获取的父类的属性和方法的访问权限

3. Getter 和 Setter

类允许使用get 和 set 方法

4. static 方法

static 类方法是在类本身上定义的

只能通过 类去调用，而不能通过类对象去调用

如果要在static方法使用类对象, 可以把类对象作为参数传到static 方法去
