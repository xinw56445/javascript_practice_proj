JavaScript 类

1. 类的语法

使用关键字 class 创建类

始终添加名为constructor() 方法

          class ClassName {
            constructor() { ... }
          }
        
类不是对象，只是javascript 对象模版

2. 类的使用方法

创建类对象

3. constructor 方法

构造方法是一种特殊的方法

  • 它必须拥有确切名称的"构造函数"
  • 创建新对象时自动执行
  • 用于初始化对象属性
  • 如果未定义构造韩式方法，JavaScript 会添加新的构造函数方法

4. class 方法

创建类方法语法

          class ClassName {
            constructor() { ... }
            method_1() { ... }
            method_2() { ... }
            method_3() { ... }
          }
