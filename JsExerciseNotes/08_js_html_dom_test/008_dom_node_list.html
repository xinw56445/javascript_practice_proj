JavaScript HTML DOM 节点列表

NodeList 对象是一个从文档中获取的节点列表（集合）

所有浏览器的 childNodes 属性返回的是NodeList 对象

部分浏览器的 querySelectorAll() 返回的NodeList 对象

NodeList 对象 length 属性

NodeList 对象有 length 属性，返回节点列表中的节点数

HTML Collection 与 NodeList 的区别

HTML Collection 是 JavaScript 集合，是基于数组的对象

NodeList 是基于对象的集合，是基于节点的对象

HTML Collection 不是 NodeList 的实例，而是类似于 NodeList 的对象

HTML Collection 元素可以通过name、id或索引来获取

NodeList 元素只能通过索引来获取

只有NodeList 对象有包含属性节点和文本节点
