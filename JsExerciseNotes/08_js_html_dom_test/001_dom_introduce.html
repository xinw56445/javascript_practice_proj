JavaScript HTML DOM

HTML DOM

当页面被加载时，浏览器会创建页面的文档对象模型（Document Object Model）

JavaScript 能够改变页面中所有HTML 元素，属性以及CSS样式

JavaScript 可以对页面中所有的事件做出反应

查找HTML 元素

  • 通过id 找到html元素
  • 通过标签名 找到html 元素
  • 通过类名 找到html元素

通过 id 查找HTML元素

var x = document.getElementById("demo");

通过标签名 查找HTML元素

var x = document.getElementsById("demo");
var y = x.document.getElementsByTagName("p");

通过类名 查找HTML元素

var x = document.getElementsByClassName("demo");
