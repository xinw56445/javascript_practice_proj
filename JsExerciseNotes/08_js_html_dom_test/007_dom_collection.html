HTML Collection 对象

HTMLCollection 对象是 HTML 元素的集合，比如 document.all 等，它是 NodeList 的子类。

getElementsByTagName() 方法返回HTML Collection对象，其类似于包含HTML 的一个数组

HTML Collection对象的 length 属性

HTMLCollection 对象有 length 属性，它返回集合中的元素数。

HTMLCollection对象有length属性, 但其绝非数组，不可使用数组方法
