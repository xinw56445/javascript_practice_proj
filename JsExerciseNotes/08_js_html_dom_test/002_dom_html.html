JavaScript HTML DOM改变HTML

改变HTML 输出流

document.write() 可用于直接向HTML 输出流写内容

document.write(Date()); // 这个方法会覆盖之前的html输出

改变HTML 内容

使用innerHTML 属性可以改变

document.getElementById("demo").innerHTML = "Hello World"; // 获取 id 为‘demo’ 的HTML元素，并且将其内容改变为‘Hello World’

改变HTML 属性

使用setAttribute() 可以改变

document.getElementById("image").setAttribute("src", "demo2.jpg"); // 或者 document.getElementById("image").src = "demo2.gif";
